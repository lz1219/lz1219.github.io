AI AgentгҖҒMCP дёҺ Skillsпјҡжһ„е»әжҷәиғҪеҠ©жүӢзҡ„ж ёеҝғжҠҖжңҜ

йҡҸзқҖеӨ§иҜӯиЁҖжЁЎеһӢзҡ„еҝ«йҖҹеҸ‘еұ•пјҢAI AgentпјҲжҷәиғҪдҪ“пјүе·ІжҲҗдёәеҪ“еүҚдәәе·ҘжҷәиғҪйўҶеҹҹжңҖзғӯй—Ёзҡ„ж–№еҗ‘д№ӢдёҖгҖӮжң¬ж–Үе°Ҷж·ұе…ҘжҺўи®Ё AI Agent зҡ„ж ёеҝғжҰӮеҝөгҖҒMCPпјҲModel Context ProtocolпјүеҚҸи®®д»ҘеҸҠ Skills зі»з»ҹпјҢеё®еҠ©дҪ зҗҶи§ЈеҰӮдҪ•жһ„е»әејәеӨ§зҡ„жҷәиғҪеҠ©жүӢгҖӮ

дёҖгҖҒд»Җд№ҲжҳҜ AI Agentпјҹ

AI Agent жҳҜжҢҮиғҪеӨҹж„ҹзҹҘзҺҜеўғгҖҒеҒҡеҮәеҶізӯ–е№¶жү§иЎҢеҠЁдҪңзҡ„жҷәиғҪзі»з»ҹгҖӮдёҺдј з»ҹзҡ„иҒҠеӨ©жңәеҷЁдәәдёҚеҗҢпјҢAI Agent е…·еӨҮд»ҘдёӢж ёеҝғиғҪеҠӣпјҡ

1.1 ж ёеҝғзү№еҫҒ

  • иҮӘдё»жҖ§пјҡиғҪеӨҹзӢ¬з«Ӣе®ҢжҲҗд»»еҠЎпјҢж— йңҖжҢҒз»ӯзҡ„дәәе·Ҙе№Ійў„
  • ж„ҹзҹҘиғҪеҠӣпјҡеҸҜд»ҘзҗҶи§Је’Ңи§Јжһҗеҗ„з§Қиҫ“е…ҘдҝЎжҒҜ
  • еҶізӯ–иғҪеҠӣпјҡеҹәдәҺзӣ®ж Үе’ҢдёҠдёӢж–ҮеҒҡеҮәеҗҲзҗҶеҲӨж–ӯ
  • жү§иЎҢиғҪеҠӣпјҡиғҪеӨҹи°ғз”Ёе·Ҙе…·гҖҒAPI жҲ–жү§иЎҢд»Јз ҒжқҘе®ҢжҲҗе®һйҷ…д»»еҠЎ
  • и®°еҝҶиғҪеҠӣпјҡдҝқжҢҒдёҠдёӢж–Үи®°еҝҶпјҢж”ҜжҢҒеӨҡиҪ®еҜ№иҜқе’Ңй•ҝжңҹи®°еҝҶ

1.2 е…ёеһӢеә”з”ЁеңәжҷҜ

  • д»Јз ҒеҠ©жүӢпјҡиҮӘеҠЁзј–еҶҷгҖҒи°ғиҜ•е’ҢйҮҚжһ„д»Јз Ғ
  • ж•°жҚ®еҲҶжһҗпјҡеӨ„зҗҶе’ҢеҲҶжһҗеӨ§и§„жЁЎж•°жҚ®йӣҶ
  • е®ўжңҚзі»з»ҹпјҡжҷәиғҪеӣһзӯ”з”ЁжҲ·й—®йўҳе№¶жү§иЎҢж“ҚдҪң
  • дёӘдәәеҠ©жүӢпјҡз®ЎзҗҶж—ҘзЁӢгҖҒйӮ®д»¶е’Ңд»»еҠЎ

дәҢгҖҒMCPпјҡжЁЎеһӢдёҠдёӢж–ҮеҚҸи®®

MCPпјҲModel Context ProtocolпјүжҳҜдёҖз§Қж–°е…ҙзҡ„еҚҸи®®ж ҮеҮҶпјҢж—ЁеңЁи§„иҢғ AI жЁЎеһӢдёҺеӨ–йғЁе·Ҙе…·е’Ңж•°жҚ®зҡ„дәӨдә’ж–№ејҸгҖӮ

2.1 MCP зҡ„ж ёеҝғд»·еҖј

MCP еҚҸи®®и§ЈеҶідәҶд»ҘдёӢеҮ дёӘе…ій”®й—®йўҳпјҡ

  1. ж ҮеҮҶеҢ–е·Ҙе…·и°ғз”Ёпјҡе®ҡд№үз»ҹдёҖзҡ„е·Ҙе…·жҸҸиҝ°е’Ңи°ғз”Ёж јејҸ
  2. дёҠдёӢж–Үз®ЎзҗҶпјҡй«ҳж•Ҳдј йҖ’е’Ңз®ЎзҗҶеҜ№иҜқдёҠдёӢж–Ү
  3. жқғйҷҗжҺ§еҲ¶пјҡзЎ®дҝқе·Ҙе…·и°ғз”Ёзҡ„е®үе…ЁжҖ§е’ҢеҸҜжҺ§жҖ§
  4. еҸҜз»„еҗҲжҖ§пјҡж”ҜжҢҒеӨҡдёӘе·Ҙе…·е’ҢжҠҖиғҪзҡ„зҒөжҙ»з»„еҗҲ

2.2 MCP жһ¶жһ„з»„д»¶

  • ClientпјҡAI жЁЎеһӢдҫ§пјҢиҙҹиҙЈеҸ‘иө·иҜ·жұӮе’Ңи§Јжһҗе“Қеә”
  • Protocolпјҡе®ҡд№үж¶ҲжҒҜж јејҸе’ҢйҖҡдҝЎи§„еҲҷ
  • Serverпјҡе·Ҙе…·жңҚеҠЎдҫ§пјҢжҸҗдҫӣе…·дҪ“еҠҹиғҪе®һзҺ°

2.3 MCP ж¶ҲжҒҜж јејҸзӨәдҫӢ

дёүгҖҒSkills зі»з»ҹпјҡжү©еұ• AI иғҪеҠӣ

Skills зі»з»ҹжҳҜ AI Agent зҡ„иғҪеҠӣжү©еұ•жңәеҲ¶пјҢйҖҡиҝҮйў„е®ҡд№үзҡ„жҠҖиғҪжЁЎеқ—пјҢи®© AI иғҪеӨҹжү§иЎҢзү№е®ҡйўҶеҹҹзҡ„еӨҚжқӮд»»еҠЎгҖӮ

3.1 Skills зҡ„и®ҫи®ЎеҺҹеҲҷ

  • жЁЎеқ—еҢ–пјҡжҜҸдёӘжҠҖиғҪзӢ¬з«Ӣе°ҒиЈ…пјҢдҫҝдәҺз»ҙжҠӨе’ҢеӨҚз”Ё
  • еҸҜеҸ‘зҺ°жҖ§пјҡAI иғҪеӨҹиҮӘеҠЁеҸ‘зҺ°е’ҢйҖүжӢ©еҗҲйҖӮзҡ„жҠҖиғҪ
  • еҸӮж•°йӘҢиҜҒпјҡзЎ®дҝқиҫ“е…ҘеҸӮж•°зҡ„жӯЈзЎ®жҖ§е’Ңе®үе…ЁжҖ§
  • й”ҷиҜҜеӨ„зҗҶпјҡжҸҗдҫӣжё…жҷ°зҡ„й”ҷиҜҜдҝЎжҒҜе’ҢжҒўеӨҚжңәеҲ¶

3.2 еёёи§ҒжҠҖиғҪзұ»еһӢ

жҠҖиғҪзұ»еһӢ еҠҹиғҪжҸҸиҝ° зӨәдҫӢ
ж–Үд»¶ж“ҚдҪң иҜ»еҶҷгҖҒжҗңзҙўгҖҒеӨ„зҗҶж–Үд»¶ иҜ»еҸ–й…ҚзҪ®ж–Үд»¶гҖҒжҗңзҙўд»Јз Ғ
зҪ‘з»ңиҜ·жұӮ HTTP/API и°ғз”Ё иҺ·еҸ–еӨ©ж°”ж•°жҚ®гҖҒжҹҘиҜўиӮЎзҘЁ
д»Јз Ғжү§иЎҢ иҝҗиЎҢд»Јз ҒзүҮж®ө Python и„ҡжң¬жү§иЎҢгҖҒSQL жҹҘиҜў
ж•°жҚ®еӨ„зҗҶ ж јејҸиҪ¬жҚўгҖҒеҲҶжһҗ JSON и§ЈжһҗгҖҒж•°жҚ®еҸҜи§ҶеҢ–
зі»з»ҹйӣҶжҲҗ дёҺеӨ–йғЁжңҚеҠЎдәӨдә’ GitHubгҖҒSlackгҖҒж•°жҚ®еә“

3.3 жҠҖиғҪе®ҡд№үзӨәдҫӢ

еӣӣгҖҒе®һжҲҳпјҡжһ„е»әдёҖдёӘз®ҖеҚ•зҡ„ AI Agent

4.1 жҠҖжңҜйҖүеһӢ

  • жЎҶжһ¶пјҡеҹәдәҺ MCP еҚҸи®®зҡ„ Agent жЎҶжһ¶
  • жЁЎеһӢпјҡж”ҜжҢҒ function calling зҡ„ LLM
  • жҠҖиғҪеә“пјҡйў„е®ҡд№үеёёз”ЁжҠҖиғҪжЁЎеқ—

4.2 ж ёеҝғжөҒзЁӢ

4.3 жіЁж„ҸдәӢйЎ№

  • е®үе…ЁжҖ§пјҡйҷҗеҲ¶ж•Ҹж„ҹж“ҚдҪңзҡ„жқғйҷҗ
  • жҖ§иғҪпјҡзј“еӯҳеёёз”Ёз»“жһңпјҢеҮҸе°‘йҮҚеӨҚи°ғз”Ё
  • еҸҜи§ӮжөӢжҖ§пјҡи®°еҪ•ж—Ҙеҝ—пјҢдҫҝдәҺи°ғиҜ•е’Ңзӣ‘жҺ§
  • з”ЁжҲ·дҪ“йӘҢпјҡжҸҗдҫӣжё…жҷ°зҡ„иҝӣеәҰеҸҚйҰҲ

дә”гҖҒжңӘжқҘеұ•жңӣ

AI Agent жҠҖжңҜд»ҚеңЁеҝ«йҖҹеҸ‘еұ•дёӯпјҢд»ҘдёӢеҮ дёӘж–№еҗ‘еҖјеҫ—е…іжіЁпјҡ

  1. еӨҡ Agent еҚҸдҪңпјҡеӨҡдёӘжҷәиғҪдҪ“еҚҸеҗҢе®ҢжҲҗеӨҚжқӮд»»еҠЎ
  2. й•ҝжңҹи®°еҝҶпјҡи·ЁдјҡиҜқзҡ„и®°еҝҶе’ҢдёӘжҖ§еҢ–еӯҰд№ 
  3. иҮӘдё»и§„еҲ’пјҡжӣҙеӨҚжқӮзҡ„д»»еҠЎеҲҶи§Је’Ңжү§иЎҢи§„еҲ’
  4. дәәжңәе…ұеҲӣпјҡдәәзұ»дёҺ AI жӣҙж·ұеәҰең°еҚҸеҗҢе·ҘдҪң

з»“иҜӯ

AI AgentгҖҒMCP еҚҸи®®е’Ң Skills зі»з»ҹе…ұеҗҢжһ„жҲҗдәҶзҺ°д»ЈжҷәиғҪеҠ©жүӢзҡ„жҠҖжңҜеҹәзҹігҖӮзҗҶи§Јиҝҷдәӣж ёеҝғжҰӮеҝөпјҢе°Ҷеё®еҠ©ејҖеҸ‘иҖ…жһ„е»әжӣҙејәеӨ§гҖҒжӣҙзҒөжҙ»зҡ„ AI еә”з”ЁгҖӮ
